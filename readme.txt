An undepended donation campaign progress bar

== Description ==

This plugin was built for a specific personal usage, and for that it is good as it is. if it will prove to be needed, i will try to improve the plugin.

Translators - [Fatcow](http://www.fatcow.com "Fatcow").

== Installation ==

Installtion is simple:

1. Upload the directory 'unpluged bar' to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

Setting is also simple:

1. Go to "unpluged bar" tab in "Plugins" menu.
2. Set the donation goal and present state.
3. Set the width of the middle part of the bar.
4. Set the link to the donation page/
3. activate the plugin by pasting the shortcode [unpbr] in the specific wanted post or page.

== Frequently Asked Questions ==
Q: Does it support RTL sites?
A: Since 0.7, unpluged-bar supports RTL and LTR direction.
